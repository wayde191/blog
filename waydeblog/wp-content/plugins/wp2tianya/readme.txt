== Description ==

同步发表 WordPress 博客日志到 天涯博客,初次安装必须设置后才能使用。

Related Links:

* <a href="http://www.hostscheme.com/125.html" title="wp2tianya Plugin for WordPress" target="_blank">Plugin Homepage</a>

Version 0.9.0 支持功能

1。支持将Wordpress中文章链接发布到天涯博客;

2。支持可选原文链接显示的位置;

3。不支持将Wordpress中私密(private)文章发布到天涯博客(未经严格测试）。


== Installation ==

1. 上传 `wp2tianya.php` 到 `/wp-content/plugins/wp2tianya/` 目录

2. 在Wordpress后台控制面板"插件(Plugins)"菜单下激活wp2tianya插件

3. 在Wordpress后台控制面板"配置(Settings)->wp2tianya"菜单下设置插件的必须信息。（只有经过设置，插件才能正常使用）

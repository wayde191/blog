== Description ==

同步发表 WordPress 博客日志到 网易博客,初次安装必须设置后才能使用。

Related Links:

* <a href="http://www.hostscheme.com/105.html" title="wp2netease Plugin for WordPress" target="_blank">Plugin Homepage</a>

Version 1.1.0 支持功能

1。支持将Wordpress中文章链接发布到网易博客;

2。支持可选原文链接显示的位置;

3。支持是否公开还是private;

4。支持是否同步发往网易微博;

5。不支持将Wordpress中私密(private)文章发布到网易博客(未经严格测试）。

6。支持在同步网易博客时带标签

== Installation ==

1. 上传 `wp2netease.php` 到 `/wp-content/plugins/wp2netease/` 目录

2. 在Wordpress后台控制面板"插件(Plugins)"菜单下激活wp2netease插件

3. 在Wordpress后台控制面板"配置(Settings)->wp2netease"菜单下设置插件的必须信息。（只有经过设置，插件才能正常使用）

== Frequently Asked Questions ==

= 1。如何填写网易博客的UR =

如网易博客地址为http://hostscheme.blog.163.com/blog，则网易博客的UR应填写godmagic，如果为中文请使用urlencode 

= 2。关于同步不成功 =

网易博客限制发文频率，频率高了以后会出现验证码。如果同步不成功，可能是频率太高所致。
